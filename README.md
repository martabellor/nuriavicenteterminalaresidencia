# Cena de despedida de Nuria

Web en Next.js para anunciar la cena de despedida de Nuria y permitir inscripciones.

## Qué incluye
- Página pública bonita con temática de laboratorio
- Menú y precio
- Cuenta atrás
- Formulario para apuntarse
- Campo de alergias
- Elección pescado o carne
- Panel admin privado
- Descarga CSV solo para admin

## Variables de entorno en Vercel

Configura estas variables:

- SUPABASE_URL
- SUPABASE_SERVICE_ROLE_KEY
- ADMIN_PASSWORD

## Base de datos

Ejecuta el contenido de `supabase-schema.sql` en el SQL Editor de Supabase.
